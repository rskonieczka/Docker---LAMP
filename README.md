# Docker - Lamp

Ten projekt ułatwia uruchomienie kompletnego środowiska LAMP (Linux, Apache, MySQL, PHP) w kontenerach Dockerowych

## Uruchomienie

```bash
cp sample.env .env
docker-compose up -d
```

### Dodatkowe polecenia

```bash
docker-compose stop #zatrzymanie
docker-compose rm -f #usuwanie
```

### Status

```bash
docker-compose ps
```
